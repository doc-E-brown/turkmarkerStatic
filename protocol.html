Mechanical Turk Facial Landmarking Task

Please ensure that you read all of these instructions prior to accepting the HIT

Description of Task

During this task you will be asked to indicate the location of 22 facial landmarks on four (4) separate images and will be required to repeat the task three (3) times for each image; thus completing the task for 12 images in total. The images will be presented to you in random order and for each image the order in which the facial landmarks are to be identified will also be randomized.

If completed correctly it is estimated that the task will take 15 - 20 minutes to complete.

Performance Bonus

A bonus payment is available for this HIT which is designed to encourage a careful and considerate approach to completing the task. Those workers who achieve an agreement level of 90% or greater with our "expert" workers will receive a bonus payment of $X for this task. A landmark position is said to be in agreement with our "expert" workers if it is positioned within a specified distance from the position chosen by the expert. A bonus payment is to be awarded if 90% or more of all the landmarks across all images and repeats are in agreement with the "experts".

User Interface

This section describes the user interface that will be used to complete the landmarking task.

The components of the user interface are as follows:

  • Image for Landmarking: this is the image that you will be processing, simply click on the location in the images corresponding to the closes position of the current specified landmark.
  • Example Image: this generic image is updated for each landmark in the set and is used to provide an indication of the approximate location of the current landmark being selected. DO NOT CLICK ON THIS IMAGE. You should also notice that the example image illustrates the point that is currently selected for positioning (see example above) and when you select a different point the image and guidance will change correspondingly.
  • Landmark Selectors: this part of the UI serves two functions: 1. to provide you with the list of landmarks to be applied to the image and the order in which they are to be applied and 2. to allow you to select each landmark to be processed. (Note you can use the f keyboard shortcut to select the next landmark in the list and the b keyboard shortcut to select the previous landmark.
  • Progress Bar: provides an indication of completion progress.
  • Button Panel: there are three (3) separate buttons in the panel:
    • Hide / Show All Landmarks: if previously placed landmarks are obstructing the view of the image to place subsequent landmarks you can click on this button or use the shortcut key h to hide / show all the landmarks except the one being placed. By default only the landmark currently being selected is shown by the system.
    • Next Image: this button will only be visible when all landmarks for the current image are in valid positions and the landmarking task has been successfully completed for the image.
    • Submit: this button will only be visible when the entire HIT has been completed and is ready for submission.

Landmark Validation

The system will check the approximate position of each landmark on the target image. If the landmark is placed in a valid position it will show green in colour, if the location is not valid the landmark will be coloured red and a message will appear in the dialogue box as shown in the image below. You will not be able to complete the task until all landmarks for all images are in valid positions
